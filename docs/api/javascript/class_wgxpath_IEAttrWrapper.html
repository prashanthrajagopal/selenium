A wrapper for an attribute node in old IE.

Note: Although sourceIndex is equal to node.sourceIndex, it is denormalized into a separate parameter for performance, so that clients constructing multiple IEAttrWrappers can pass in the same sourceIndex rather than re-querying it each time.
