code »bot.storage.getLocalStorage (opt_window) ⇒ !bot.storage.Storage

A factory method to create a wrapper to access the HTML5 localStorage object. Note: We are not using Closure from goog.storage, Closure uses "window" object directly, which may not always be defined (for example in firefox extensions). We use bot.window() from bot.js instead to keep track of the window or frame is currently being used for command execution. The implementation is otherwise similar to the implementation in the Closure library (goog.storage.mechansim.HTML5LocalStorage).

Parameters
opt_window: Window=
The window whose storage to access; defaults to the main window.
Returns
