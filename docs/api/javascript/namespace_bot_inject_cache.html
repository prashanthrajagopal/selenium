code »bot.inject.cache.CACHE_KEY_ : string

The property key used to store the element cache on the DOCUMENT node when it is injected into the page. Since compiling each browser atom results in a different symbol table, we must use this known key to access the cache. This ensures the same object is used between injections of different atoms.
