code »bot.Keyboard.Key.fromChar (ch) ⇒ {key: !bot.Keyboard.Key, shift: boolean}

Given a character, returns a pair of a key and a boolean: the key being one that types the character and the boolean indicating whether the key must be shifted to type it. This function will never return a numpad key; that is, it will always return a symbol key when given a number or math symbol. If given a character for which this module does not know the key (the key is not in the bot.Keyboard.Keys enumeration), returns a key that types the given character but has a (likely incorrect) keycode of zero.
