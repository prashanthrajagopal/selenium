Finds an element by using the given lookup strategy.

Parameters
strategy: string
The strategy to use to locate the element.
using: string
The locator to use.
opt_root: ?{ELEMENT: string}=
The WebElement reference for the element to perform the search under. If not specified, will use document for the target page.
opt_window: {WINDOW: string}=
The serialized window object for the page to find the element in. The referenced window must exist in the page executing this script's cache.
